O que é o VisualG

VisualG é uma ferramenta didática que auxilia no ensino de algoritmos através de uma linguagem próxima ao português estruturado.

Finalidade e Público-alvo

Destinado a iniciantes no mundo da programação, o VisualG ajuda a compreender os fundamentos da lógica algorítmica.

Ambiente de Desenvolvimento

  • Interface intuitiva
  • Editor de código, console e botões de execução
  • Execução passo a passo e mensagens de erro didáticas

Tipos de Variáveis

  • Inteiro (int): Ex: 10, -5, 0
  • Real (real): Ex: 3.14, -2.5, 0.0
  • Caractere (char): Ex: 'a', '!', ' '

Exemplo de Algoritmo

// Calcula a média de dois números
algoritmo "media"
var
  num1, num2, media: real
inicio
  escreva("Digite o primeiro número: ")
  leia(num1)
  escreva("Digite o segundo número: ")
  leia(num2)
  media := (num1 + num2) / 2
  escreval("A média é: ", media)
